## The map

- **`lessons/NN-short-name/`** — the lessons. Each is a `LESSON.md` (the story) + `install-prompt.md` (agent-driven setup), plus any payload files. **Full annotated index: [`README.md`](../README.md)** — start there to find a lesson by topic.
